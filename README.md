# Tech Study Landing Page

Landing page moderna que consome a API desenvolvida no Módulo 4 e apresenta cursos e materiais de estudo.

## Tecnologias
- Next.js
- React
- TypeScript
- TailwindCSS
- Axios
- Framer Motion

## Como executar

```bash
npm install
npm run dev
```
